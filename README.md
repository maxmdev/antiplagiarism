# antiplagiarism
Simple anti-plagiarism detection tool with local library

# Implemented methods:
1. Shingles
2. Cosine Similarity
3. Moodle Crot
4. Levenshtein Distance
5. TF*IDF

# Requirements
pip install -r requirements.txt.

# Starting
Run plag.py to start.
